## [烧饼排序](https://leetcode-cn.com/problems/pancake-sorting/comments/)
假设盘子上有n块面积大小不一的烧饼，你如何用一把锅铲进行若干次翻转，让这些烧饼的大小有序(小的在上，大的在下)

```python
res = []
def pancake_sort(cakes):
    sort_cake(cakes, len(cakes))
    return res

def sort_cake(cakes, n):
    if n == 1: return
    max_num = 0
    max_idx = 0
    for i in range(n):
        if cakes[i] > max_num:
            max_num = cakes[i]
            max_idx = i
    reverse(cakes, max_idx)
    res.append(max_idx + 1)
    reverse(cakes, n - 1)
    res.append(n)
    sort_cake(cakes, n - 1)

def reverse(cakes, n):
    cakes = cakes[:n + 1][::-1] + cakes[n + 1:]
```

## 前缀和
给定一个整数数组和一个整数k，你需要找到该数组中和为k的连续的子数组的个数。

```python
def pre_sum(nums, k):
    sum_list = Counter()
    sum_list[0] = 1
    temp = 0
    res = 0
    for i, num in enumerate(nums):
        temp += num
        sum_list[temp] += 1
        res += sum_list[temp - k]
    return res
```
